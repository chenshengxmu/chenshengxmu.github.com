title: Tesla下一代：Model 3的前情与展望

最近Tesla的一大新闻，是宣布了下一款新车的名字——Model 3。而且CEO穆斯克向媒体“哭诉”，Model 3 并不是他的首选，无奈被强权所迫改了名字。

Tesla通常是怎么给自己的车型起名字呢？命名规则一般是 Model+字母的形式。目前 Tesla 唯一现车销售的车型是 Model S，字母代表的是“Sedan”。即将上市的 Model X，字母代表的则是“Crossover”。而这款刚确认名字的3.5万美元电动车，曾被外界广泛称为 Model E，字母代表了“Electric”或“Economic”。不过之前官方从来没确认过这个名字。

按照穆斯克的说法是，某天一个朋友问他，下一款车型打算叫什么呢。他回答说，既然我们有了S和X，那说不定会来个E。这就是 Tesla“SEX”组合的由来。

于是去年8月，Tesla 向美国专利商标局申请了“Model E”商标。

2000年福特曾起诉过一家叫“Model E”的网络公司，理由是这名字跟福特传奇的“Model T”相似度太高。不过法官驳回了福特的诉讼。福特还在2011年起诉过F1法拉利车队，认为车队将赛车命名为 F150 侵犯了福特 F 系列皮卡的商标权。最终双方达成和解，法拉利将赛车更名为 Ferrari 150°Italia。

在Tesla申请“Model E”后，当时福特的新闻发言人 Jay Cooney 表示，福特并不介意Tesla注册这个商标。但当天他又改变了说法，表示福特将会认真评估这件事，不便给予更多置评。

穆斯克透露，福特对 Tesla 申请注册 Model E 商标很不满意，威胁说要起诉他。“福特杀死了‘SEX’！”，穆斯克告诉媒体。

不过福特的确有个E系列车型，是大型厢式货车。1960年代就上市的E系列将于今年停产，被福特全顺所取代。

但在我看来，这次更名事件对 Tesla 利远大于弊。穆斯克也很好地抓住了这次机会，让全世界都关注了 Tesla 的全新车型。

其实在Tesla申请 Model E 商标之前，外界都把这款低价电动车叫做 Gen III，意为这家公司的第三代车型。Tesla 的第一代车型是在莲花跑车架构上开发的 Roadster，随后就是著名的 Model S 和 Model X 了。

Model 3平台并不是一个缩小版的 Model S，它是由 Tesla 工程开发副总裁 Chris Porritt 负责的全新平台。这哥们之前是 Aston Martin 的工程开发负责人。

按照穆斯克公布的时间表，Model 3将在2016年发布，于2017年开始正式销售。最近穆斯克刚去了趟英国，宣布要在英国建一个研发中心，据说这个研发中心就是为 Chris Porritt 和 Model 3 准备的。（微信公号「KnewOneCom」中回复「M3」，浏览穆斯克讲解 Model 3 电池和续航里程的视频）

Model 3 对穆斯克和他的公司来说，都意义非凡。所以没有沿用之前的命名规则也没什么大不了的。穆斯克曾在博客中公布了 Tesla 的发展规划：

2. 利用这辆跑车吸引的资金，开发出一辆更多人能买得起的电动车；

4. 推进以上三步的同时，提供零排放的电力能源；

根据这个发展规划来看，Model 3 代表了 Tesla 发展的第三个里程碑，也是 Tesla 能否真正取得成功的最后一步。所以数字“3”颇具意义。

Model 3 承载的责任，不仅仅是让大众消费者接受纯电动车，它还要改变普通民众消费能源的方式。Tesla 的最终成功，对穆斯克的另一家上市公司Solarcity也很重要。

Tesla 会在自己的充电、储能业务中使用 Solarcity 的产品和技术，而 Solarcity 也会在自己的业务中使用 Tesla 的电池技术。如果你不认同我的“阴谋论”，添加我的个人微信「yuxinlie」来讨论。

从这个意义上说，采用新技术、扮演新角色的 Model 3 理应采用与 Model S、X 不同的命名方式。要知道 Tesla 的前两代产品命名规则也是各不相同的。Roadster 跑车之后的第二代产品，才开始采用新的命名规则。

而说到“SEX”产品组合被击碎这个事儿，其目的原本就是讲一个故事，吸引大众对 Tesla 车型的注意力。那么“被传统汽车巨头福特用强权威胁、被迫采用全新名字”的故事，会产生更好的传播性和影响力。而另一方面，穆斯克在某种程度上巧妙地保留了 Tesla 的“SEX”组合。因为在极客圈里，很多人用“S3X”来代表“SEX”。
