title: 特斯拉对特斯拉：Model X 和 Model S，谁更强？
---
Tesla 的电动 SUV 车型 Model X 是该公司今年主打的一款车型，除了一些设计上的亮点之外，Model X 的性能是许多人关心的焦点。近日，汽车媒体 DragTimes 进行了一项测试：将 Model X P90D 和 Model S P90D 放在一起比赛，看看它们在直线跑道上的性能如何。结果 Model S P90D 以微弱优势获胜。

两辆车的动力总成完全相同，并在测试中均使用了“狂暴模式”(Ludicrous Mode)。在四分之一英里长的直线跑道中，Model X 耗时?11.61 秒，0 到 60 英里/小时加速?3.1 秒，比官方数据还低了 0.1 秒。考虑到直线加速受到包括车手体重在内多方面因素的影响，这个成绩合情合理。

在多轮较量中，Model X 的成绩位置在?11.6 秒左右，而 Model S 的成绩维持在?11.3 至 11.5 秒之间。考虑到 Model X 车更重以及 Model S 更好的空气动力学性能，DragTimes 认为两辆车的性能不分伯仲，并表示 Model X 的动力输出可能更强。

不过 DragTimes 也指出，测试发现，和 Model S 相比，Model X 会显著地消耗更多的电能。数据读表显示，测试后的 Model X 使用了?28.3 kw/h 的电能，Model S 消耗的能量则是 22.7 kw/h。
