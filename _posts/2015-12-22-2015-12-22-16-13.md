作者注：“动见”原本是前任虎嗅编辑祁萌同学的明星专栏，最近由我尝试接替写作，大概每周一更，还是从软件、硬件、投资等方向观察总结一周移动互联网值得关注的动态和趋势。

今年1月26日，联想在北京国家会议中心对中国市场发布新Moto X、Moto X pro、新Moto G三款手机，已经成为联想旗下品牌的Moto终于重返中国。

Moto的研发实力和品牌价值无需多言。如今1年将尽，联想有打好手里这张王牌吗？

在研发方面，联想对Moto的整合已见成效，比如今年11月16日发布的乐檬X3 就采用了Moto的“丽音”技术，联想移动高管在Moto X 极发布会后接受采访时也曾透露，联想品牌的手机在语音通话质量上得益于Moto的技术积累。

联想移动总裁陈旭东在提到Moto X 极的防摔屏幕时还说了这么一段话：“这种手机给大家想一年的时间也做不出来，我研究三年了，我已经做好产品了，你可以解剖我的东西，也需要很多工程化，因为这款手机不是在OEM生产了，是我自己的工厂生产的，所有的技术掌握在自己手里。”

在品牌方面，联想集团副总裁、手机中国区总经理杨峻曾在公开场合讲过联想手机的品牌整合策略，未来的手机都在联想乐檬和摩托罗拉两个品牌下推出，乐檬产品定位在3000元以下，而摩托罗拉则从1500元到5000元。简而言之，联想以后在3000以上的高端区间全靠Moto。

目前看来，如果想要挑刺，联想对Moto品牌的运营还有很多混乱和疏漏之处，这里略具几例，并尝试纸上谈兵，用键盘给联想提些建议。

前几天联想刚刚发布了旗舰机型Moto X 极，主打摔不碎的屏幕，价格5288元起，其实就是Droid Turbo 2和Moto X Force的国行版。据联想移动总裁陈旭东透露，这款机器于10月28日在美国发布之后，销量已经接近百万。中国手机市场竞争更激烈、消费者更讲究性价比，联想在这里发布这款机器，并定价于5千之上，可能更多是为品牌而非销量考虑。

首先是与其他的产品不统一，根据Moto中文官网，正式引入中国的手机型号还包括Moto X Pro（即Nexus6 的国行版本）、Moto G、Moto X、Moto X Style（即Moto X Pure Edition），其中Moto X Style仍保留英文名称，并未汉化，偏偏只有Moto X Force 有了中文名。

其次是没有把握消费者心理，Moto曾经也有非常经典的“明”系列手机，可算国际品牌本土化运作的优良范例，但当时Moto尚未被联想收购，如今情势已然不同，联想手机品牌积淀不如Moto，一个弱中国品牌（相对而言）收购一个强外国品牌后，消费者最担心的就是被收购的强品牌质量下滑、设计不如往昔，为减少这样的担忧，理论上最简单直接的策略就是尽量减少动作，能不动就不动，既然是同一款机型，在中国也叫Moto X Force就好了，何必多此一举弄个“Moto X 极”。

而且“X 极”这个的谐音似乎很容易被恶搞，这里放一条前几天站内相关稿件底下的评论：

（读者评论，还有两位同学点了赞......）

其实Force一词简单好记，顺便还能和《星战》搭上边，有兴趣的同学可以看看友站PingWest这篇稿子下面的评论，好几位读者留言将“Moto X Force”解读为“Moto X 原力”，顺便说一句，友站这篇稿子在标题里就吐槽了“Moto X 极”这个名字......

其实，在这个时代，本土化和中文名已经没有绝对联系，国人对苹果生造的“iPhone”和“iPad”等等词汇一点都不抵触，微软的热门产品Xbox、Surface也从来都没有中文名称。

漂亮的实体店本身是一种价值，如果你在北京，你可以对比下西单大悦城一层苹果旗舰店和楼上苹果授权店的人气，这两者卖的产品理论上是一模一样的，大家还可以简单对比下自己每年去苹果旗舰店的次数和每年买苹果产品的次数。

结论很简单，从销售提升的角度，同样的商品，消费者更愿意去漂亮的店里购买，从品牌建设的角度，就算不买，只要店够漂亮，大家也愿意逛街的时候顺带去转转。

（Moto店正面，图片来源：Moto官方博客）

木质的moto logo招牌很有质感，略显复古，像是时装店，和苹果、微软旗舰店风格全然不同。

（Moto店内部，图片来源：Moto官方博客）

Engadget评论这间店铺与苹果旗舰店恰成对照（antithesis），苹果店让人有冷峻之感，而Moto店则更热情，或许因为Moto店更多使用暖色，木头、皮革等材质也带来了实体感。更多照片可以看我们之前的这篇文章。

但根据The Verge的说法，Moto并没有计划自建零售渠道（retail chain），这间漂亮的Moto体验店可能只是个一次性试验（one-off experiment）。

强烈建议联想不要浪费了如此出彩的店面和陈列设计，将这样的Moto体验店搬到中国来。

我司名编辑涵詔曾说，联想完全可以用最新的配置和工艺复刻经典的ThinkPad型号，估计能秒杀现在一众笔记本。

ZUK CEO 常程曾在一次群访中透露，ZUK内部有过复刻联想经典机型K900的想法。

对联想而言，要唤起大家对Moto美好记忆、引发品牌关注，有一个很简单直接的办法，复刻Moto的那些经典型号，比如三防的Defy（戴妃）、能滑出全键盘的Milestone（里程碑）等等。

（Moto的经典机型Milestone，图片来源：Wikipedia）

在很多手机不是像iPhone就是像Sony的今天，看看Moto的这些经典型号，真有恍如隔世又耳目一新之感。

这已经不是PC时代，只要靠运营，加上渠道和品牌的优势就能取胜。现在需要的竞争力已经完全不同。我去年跟你们说了几次，要醒一醒，我甚至还说了你们拿榔头敲都敲不醒，你们太慢了，在错失机会。当然，从今年年初开始看到了一些变化的迹象，比如做精品，建on line模式，上微博等等。但是肯定不够，不够到位，也不够快。

上面的文字自然算不上“榔头”，顶多算键盘侠用键盘敲了敲联想移动，我们都希望看到一个更开放、行动更迅速的联想，也希望能看到Moto成为一张没有被浪费的王牌。
